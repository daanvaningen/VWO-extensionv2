This chrome extension is build by ClickValue. ClickValue is an Amsterdam based, data driven, online marketing agency. The tool makes users able to see active VWO experiments and change test variations. For more information you can visit www.clickvalue.nl

This new version uses the VWO API to retrieve active experiments and goals.

API key

To use the API you will first need to create an API token (if you already have a token for the extension in your VWO main account you don't have to create a new one and add it directly to the extension). The token will be used to communicate with the VWO API and retrieve the data that belongs to account(s) holding the token. We advice you to create 1 token for your main account. To create a token please follow these steps:

  • Visit http://app.vwo.com/#/developers/tokens.
  • Login to the account for which you want to create a token
  • Press "Add another API token"
  • Insert a token name and as a permission select browse
  • Press "Generate"
  • Copy the token text and add it to the extension
